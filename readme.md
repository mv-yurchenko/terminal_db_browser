# Terminal DataBase Client

## Installation 

```bash
$ git clone https://github.com/mv-yurchenko/terminal_db_browser

$ cd terminal_db_browser

$ pip3 install -r requirments.txt
```

## Usage

```bash
$ python3 db_browser.py sqlite_example.db
```

## DataBases

### Supported DataBases: 
* SQLite

### Will be supported: 

* MySQL 
* ClickHouse
* Porstage SQLa
